# NAME

Pod::Utilities - Set of helper functions to ease working with Pods!

# SYNOPSIS

```perl6
use Pod::Utilities;
```

# DESCRIPTION

Pod::Utilities is a set of tools to deal with Pod elements. It lets you create several Pod objects, obtain gists and modify headings.
